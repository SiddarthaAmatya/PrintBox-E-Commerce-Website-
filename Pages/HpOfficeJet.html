Specificaiton

The HP OfficeJet Pro 9015 is an all-in-one inkjet printer that blends speed and versatility, making it suitable for small to medium-sized offices. It offers fast printing, automatic two-sided printing, and a 35-page automatic document feeder. It has excellent print quality for both text and graphics. The printer is designed with smart tasks and workflow shortcuts, saving time in document organization. It also supports HP's Instant Ink program, which can help save on ink costs.

  • Print Technology: HP Thermal Inkjet
  • Connectivity Options: Wi-Fi, Ethernet, USB, HP Smart app, Apple AirPrint, Wi-Fi Direct Printing
  • Functions: Print, Scan, Copy, Fax
  • Print Resolution: Up to 4800 x 1200 optimized dpi (color), Up to 1200 x 1200 rendered dpi (black)
  • Print Speed: Up to 22 ppm (black and color)
  • Paper Handling: 250-sheet input tray, 60-sheet output tray, 35-sheet ADF
  • Display: 2.65" color touchscreen
  • Dimensions: 17.3" x 13.5" x 10.9"
